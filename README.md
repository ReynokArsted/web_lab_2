# web_lab_2

Это проект приложения с реактивными web-клиентом и сервером.
Согласно заданию для варианта 2, клиент принимает координаты двух точек и передаёт их серверу для вычисления расстояния между этими точками**.
Запросы и ответы для них используют формат .json.

Эндпоинты:
1. POST: /client/distance - получение расстояния между двумя точками по координатам (в примере 1 ниже показан формат запроса);
2. POST: /client/distances - получение расстояния между несколькими парами точек по координатам (в примере 2 ниже показан формат запроса);
3. POST: /server/distance - выполнение той же функции, что и у /client/distance (это прямое обращение к серверу без web-клиента);
4. POST: /server/distances - выполнение той же функции, что и у /client/distances (это прямое обращение к серверу без web-клиента).

Примеры запросов:

1.

```json
{
    "x1": 2,
    "y1": 42,
    "x2": 10,
    "y2": 10
}
```

2.

```json
[
    {"x1": 110, "y1": 0, "x2": 3, "y2": 4},
    {"x1": 10, "y1": 7, "x2": 4, "y2": 5},
    {"x1": 20, "y1": 2, "x2": 5, "y2": 6}
]
```

Примеры ответов от сервeра:

1.

```json
{
    "distance": 57.0087712549569
}
```

2.

```json
[
    {
        "distance": 107.07474025184464
    },
    {
        "distance": 6.324555320336759
    },
    {
        "distance": 15.524174696260024
    }
]
```

** Для первого запуска достаточно клонировать репозиторий с проектом
