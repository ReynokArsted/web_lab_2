# web_lab_2

Это проект приложения с реактивными web-клиентом и сервером.
Согласно заданию для варианта 2, клиент принимает координаты двух точек и передаёт их серверу для вычисления расстояния между этими точками.
Запросы и ответы для них используют формат .json.

Эндпоинты:
1. POST: /client/distance - на него отправляется json-файл (по примеру запроса 1 ниже) с координатами одной пары точек (в ответ сервер возвращает json-файл с рассчитанным расстоянием между точками);
2. POST: /client/distances - на него отправляется json-файл (по примеру запроса 2 ниже) с координатами нескольких пар точек (аналогично в ответ сервер возвращает json-файл с рассчитанными расстояниями между парами точек);
3. POST: /server/distance - прямое обращение к серверу без web-клиента (подобно /client/distance);
4. POST: /server/distances - прямое обращение к серверу без web-клиента (подобно /client/distances).

Примеры json-файлов для запросов:
1. {
    "x1": 2,
    "y1": 42,
    "x2": 10,
    "y2": 10
   }
2. [
    {"x1": 110, "y1": 0, "x2": 3, "y2": 4},
    {"x1": 10, "y1": 7, "x2": 4, "y2": 5},
    {"x1": 20, "y1": 2, "x2": 5, "y2": 6}
   ]

Примеры ответов от сервара:
1. {
    "distance": 57.0087712549569
   }
2. [
    {
        "distance": 107.07474025184464
    },
    {
        "distance": 6.324555320336759
    },
    {
        "distance": 15.524174696260024
    }
   ]
